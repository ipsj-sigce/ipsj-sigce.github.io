# 情報処理学会 コンピュータと教育研究会

Web サイトは [ipsj-sigce.github.io](https://ipsj-sigce.github.io) をご覧ください。

## （詳しい人向け）開発環境について

[mkdocs-material](https://squidfunk.github.io/mkdocs-material/reference/) を使用しています。ローカルで実行する場合は、下記のコマンドでプレビューを立ち上げることができます。

```
pip3 install -r requirements.txt
python3 -m mkdocs serve
```
