# 研究会への登録

## 登録すると

    - 該当年度の研究発表会参加費が無料になります．
    - [CE研の研究報告](http://www.bookpark.ne.jp/cm/ipsj/select_signotes2.asp?category2=CE)を全文閲覧できます．

## 登録方法

### 情報処理学会正会員の場合

- [研究会登録](http://www.ipsj.or.jp/09sig/kenkyukai/toroku.html)を行って下さい．
- [正会員登録手続き](https://www.ipsj.or.jp/02moshikomi/event/toroku.html)
  - 登録費は年度により異なりますが，年間2,500～3,000円程度です．
  - 毎年1月頃，翌年度の研究会登録の案内があります.

### 情報処理学会正会員でない場合

- できるだけ[情報処理学会へ入会](http://www.ipsj.or.jp/06mem/nyukai/nyukai.html)してください．
- 活動がCE研に限定される場合は，登録費が低廉な[準登録制度](https://www.ipsj.or.jp/02moshikomi/event/toroku-jk.html)もあります．
